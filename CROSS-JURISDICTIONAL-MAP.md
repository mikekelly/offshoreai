# Cross-Jurisdictional Map — Offshore AI

## What this is

A high-level map of how the six offshore jurisdictions
in this corpus (Jersey, Guernsey, Isle of Man, Cayman,
BVI, Bermuda) compare across the principal structuring
dimensions. Each row links to the substantive concept
files.

This is **not** a substitute for the jurisdiction-
specific content — it's a navigation surface and a
landing-page-style demonstration of the corpus's
comparative depth.

## Decision frameworks

### "Where should I form my fund?"

| Profile | Lean toward |
|---|---|
| Closed-ended PE / VC with global LP base | **Cayman ELP** + Private Funds Act 2020 |
| Hedge fund with institutional LPs | **Cayman Mutual Funds Act** (s.4(4)) |
| Emerging manager, sub-$100m, BVI-friendly | **BVI Approved Manager + Approved Fund** |
| EU-marketing-focused fund (NPPR) | **Jersey JPF** or **Guernsey PIF** |
| Family-office investment pool | **Guernsey PIF Route 2** or **Jersey JPF** |
| Multi-strategy single-entity | **Cayman SPC** or **Guernsey PCC** |
| Insurance / ILS | **Bermuda SAC** with Class 3 / 3B SPI |

### "Where should I form my trust?"

| Profile | Lean toward |
|---|---|
| Maximum forced-heirship defence | **Jersey** (Article 9(4)) |
| Family-business succession with disengaged trustee | **BVI VISTA** |
| Orphan SPV / no-beneficiary purpose trust | **Cayman STAR** |
| EU-proximate trust (cost / familiarity) | **Guernsey** or **Jersey** |
| Trust company + cell structure | **Guernsey PCC / ICC** |
| Standard discretionary family trust | **Jersey** or **Guernsey** |
| Combined with image-rights register | **Guernsey** |

### "Where should I form my (re)insurance company?"

| Profile | Lean toward |
|---|---|
| Pure single-parent captive | **Bermuda Class 1** or **Guernsey PCC cell** |
| Commercial cat reinsurance, $100m+ capital | **Bermuda Class 4** |
| Healthcare captive | **Cayman Class A-E** |
| European-parent captive | **Guernsey PCC** |
| ILS / sidecar / cat-bond series | **Bermuda SPI / SAC** |
| Life assurance (UK HNW market) | **IoM Class A-E** |

### "Where should I form my listing-vehicle TopCo?"

| Profile | Lean toward |
|---|---|
| US-listed tech holdco | **Cayman Exempted** (the Alibaba pattern) |
| UK-listed (LSE Main / AIM) | **Jersey Co (CJL 1991)** or **Guernsey** |
| Insurance / reinsurance listing | **Bermuda Exempted** |
| Lower-cost smaller listing | **BVI BC** (some exchange-acceptance limits) |
| Specialty / specific UK regulator interaction | **Jersey** with FCA dialogue |

### "Where should I plan my international wealth?"

The classic mix:

- **Jersey trust** as the central wealth wrapper (Article 9
  firewall, Article 9A reserved powers, deep
  jurisprudence);
- **Cayman / BVI** underlying companies for specific
  asset classes (US blocker for US equities; BVI BC for
  Caribbean villa);
- **IoM portfolio bond** for UK-resident family members'
  tax-efficient investment;
- **Guernsey Image Rights** if the family has
  commercially-valuable personalities;
- **Bermuda captive** for family-specific insurance
  needs.

Cross-jurisdictional structures combining multiple
domiciles are routine for HNW families.
