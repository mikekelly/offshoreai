# What Counts as "Development"

## The statutory definition

The **Planning and Building (Jersey) Law 2002** defines
"development" as:

> The carrying out of **building, engineering, mining or
> other operations** in, on, over or under land, **and**
> the making of any **material change in the use** of any
> building or other land.

Two limbs: physical operations + change of use.

## Physical operations

This covers virtually any tangible work on land:

- **Building** — new structures, extensions, conservatories;
- **Engineering** — earthworks, drainage, retaining walls;
- **Mining** — extraction, quarrying;
- **Other operations** — demolition, hardstanding,
  creating access, removing hedges, advertisements,
  scaffolding (in some contexts).

If you can see it — or it changes the physical state of
land — it's likely an "operation."

## Material change of use

Changes of use can trigger development even where no
building work occurs. Examples:

- **Residential to short-term let** (e.g. AirBnB);
- **Residential to commercial** (running a B&B from a
  home);
- **Office to residential conversion** (and vice versa);
- **Agricultural to residential** — typically requires
  permission;
- **Splitting a dwelling into two** (or merging two into
  one);
- **Class change** within commercial use (e.g. shop to
  café — may or may not be material).

"Material" means **substantive** — not every minor change
of use. A house used partly for home-office work is
typically not a change of use. A house run as a serviced
let business probably is.

## What's NOT development

- **Internal alterations** to an unlisted building —
  generally outside the planning definition;
- **Repair and maintenance** in the ordinary sense;
- **Use of a single dwelling-house** for purposes
  incidental to its primary residential use;
- **Use of agricultural land** for agriculture;
- **Use within the same "use class"** as the existing
  authorised use (where the Use Classes Order applies).

## Use Classes

Jersey uses a system of **use classes** grouping similar
uses. Movement **within** a class is typically not
development; movement **between** classes typically is.

Indicative use classes:

| Class | Examples |
|---|---|
| Class A | Shops |
| Class A2 | Financial / professional services |
| Class A3 | Restaurants, cafés |
| Class A4 | Pubs |
| Class A5 | Hot food takeaway |
| Class B1 | Office / light industrial |
| Class B2 | General industrial |
| Class B8 | Storage / distribution |
| Class C | Residential |
| Class C2 | Care homes |
| Class D | Hotels, leisure |

A change of use within a class is usually not material.
Between classes — typically requires permission.

## Borderline cases

Common questions:

### Garden office / home working

- **Incidental** to residential use (a desk in a spare
  room) — not development;
- **Substantial** (a permanent business operation with
  staff visiting) — likely development.

### Short-term lets / AirBnB

- **Occasional** lets — typically not material;
- **Regular / commercial** lets — likely a material
  change of use;
- Jersey has been **tightening** rules in this area;
- Engage planning advice before scaling up short-let
  activity.

### Annexe / granny flat occupied independently

- **Use as ancillary** to the main dwelling — typically
  not separate development;
- **Independent let** to a third party — typically
  requires permission as splitting one dwelling into two.

### Vehicle parking on garden

- **Domestic vehicles** — typically not material;
- **Commercial vehicle parking** — typically requires
  permission.

### Outdoor advertising

- **Modest signage** for a residential property — often
  permitted under specific provisions;
- **Commercial signage** — typically requires advertising
  consent.

## Why the definition matters

If your works fall within "development," you need either:

- **Permitted-development cover** under the GD Order
  2011; or
- **Planning permission** by formal application.

Works that are not "development" don't need planning
permission at all (though may still need building approval
or other consents).

## Practical recommendation

Engage with the Department's pre-application service or
take planning advice if you're unsure whether your work
falls within the definition. The cost of a Certificate of
Lawful Development (or planning application if needed)
is much less than the cost of enforcement against
unauthorised development.
