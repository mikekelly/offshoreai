# Planning and Building — Jersey

## What this section is about

Jersey planning and building law — the statutes, orders,
process and forums that control development of land in
Jersey. Every Jersey homeowner thinking about an extension,
every developer with a site, every owner of a listed
building, and every neighbour responding to a proposal
needs to navigate this regime.

The principal statute is the **Planning and Building
(Jersey) Law 2002** (PBL 2002). It works together with:

- **Planning and Building (General Development) (Jersey)
  Order 2011** — the "permitted development" exemptions;
- **Building Bye-Laws (Jersey) 2007** — separate technical
  building standards;
- **The Island Plan** — the policy framework against which
  applications are tested.

The principal decision-maker is the **Department for the
Environment**, with the **Planning Committee** considering
the more significant applications and the **Minister for
the Environment** acting on departures from the Island
Plan.

## At a glance

| Topic | Position |
|---|---|
| Principal statute | Planning and Building (Jersey) Law 2002 |
| Permitted development | General Development (Jersey) Order 2011 |
| Building standards | Building Bye-Laws (Jersey) 2007 (separate from planning) |
| Policy framework | The Island Plan (currently 2022 Island Plan, periodically refreshed) |
| Decision-maker | Department for the Environment (officer-level); Planning Committee (politicly-significant); Minister for the Environment (departures from Island Plan) |
| Standard timeline | 8 weeks for minor applications; 13 weeks for major |
| Public notice | 21 days on-site + Jersey Gazette listing |
| Consultation period | 21 days for objections |
| Appeal window | 28 days post-decision |
| Penalty for unauthorised development | Criminal: unlimited fines, possible imprisonment |
