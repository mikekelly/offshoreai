# Squeeze-Out (Part 18) vs Scheme — Jersey Target

## The question

You're acquiring a **listed Jersey company** and want 100%
ownership. **Squeeze-out** under Part 18 or **scheme of
arrangement** under Article 125 — which?

## The short answer

| Feature | Squeeze-out (Part 18) | Scheme (Article 125) |
|---|---|---|
| Initiator | Bidder | Target / bidder via target |
| Threshold | 90% acceptance | 75% in value + majority in number per class |
| Court involvement | Optional (dissenting only) | Mandatory (sanction) |
| Friendly required | No | Yes (target cooperation) |
| Effect | Compulsory acquisition of holdouts | Binds all shareholders |
| Outcome certainty | Conditional on 90% | High if approved |
| Speed | Slower (tender + 6-week notice) | Slower (court hearings) |
| Cost | Lower | Higher |

**Choose squeeze-out** for hostile / bidder-initiated bids;
**scheme** for friendly negotiated transactions with cooperating
target.

## The proper answer

### Squeeze-out (Part 18)

#### Process

1. **Tender offer** to all shareholders (Article 116);
2. **Achieve 90% acceptance** of shares to which offer relates;
3. **Squeeze-out notice** to dissenting minority (Article 117);
4. **6-week window** for dissenting shareholders to apply to
   Court (Article 121);
5. **Compulsory acquisition** if no successful court application
   (Article 118).

#### Strengths

- **Bidder-initiated** — doesn't need target cooperation;
- **No court sanction** required (in normal case);
- **Lower cost**;
- **Clear thresholds** at 90%.

#### Weaknesses

- **90% threshold uncertain** — depends on acceptance levels;
- **Dissenting-shareholder appeals** can delay;
- **Multiple-class** complexity if shares split into classes.

### Scheme of arrangement (Article 125)

#### Process

1. **Convening hearing** before Royal Court;
2. **Class meetings** with explanatory statement (Article 126);
3. **75% in value + majority in number** approval per class;
4. **Sanction hearing** before Royal Court;
5. **Court sanctions** if statutory requirements met and scheme
   is fair;
6. **Filing** of Court Act with Registrar — scheme takes
   effect.

#### Strengths

- **High certainty** if approval reached and sanction granted;
- **100% acquisition** through single instrument;
- **Binds all** including dissenters;
- **Familiar** to LSE / takeover community.

#### Weaknesses

- **Target-led** typically — needs board cooperation;
- **Court involvement** mandatory — time and cost;
- **Class composition** litigation risk;
- **Two court hearings** (convening + sanction).

### Comparison in detail

#### Threshold

- **Squeeze-out**: 90% of shares to which the offer relates —
  high threshold;
- **Scheme**: 75% in value + majority in number per class —
  more achievable for split investor bases.

#### Friendly vs hostile

- **Squeeze-out**: works either way (tender offer can be
  hostile);
- **Scheme**: needs target cooperation typically.

#### Speed

- **Squeeze-out**: tender period (typically 21–28 days);
  acceptance reach; 6-week squeeze-out window. Total: 2–4
  months.
- **Scheme**: convening hearing → class meetings → sanction
  hearing. Total: 3–4 months typically.

#### Cost

- **Squeeze-out**: lower legal cost;
- **Scheme**: higher legal cost (two court hearings, complex
  documentation).

#### Outcome certainty

- **Squeeze-out**: 90% threshold not guaranteed; partial
  acceptance leaves bidder short of 100%;
- **Scheme**: if sanctioned, achieves 100% in one step.

### When to choose squeeze-out

- **Bidder-initiated** with no target cooperation;
- **Confident** of >90% acceptance;
- **Multiple competing bidders**: tender offers more flexible;
- **Hostile bids**.

### When to choose scheme

- **Friendly negotiated** with target board cooperation;
- **Split investor base** below 90% likely acceptance;
- **Need** for certainty of 100% outcome;
- **Class-rights** complications addressable through scheme
  flexibility.

### Combination — recommended takeover

Some takeovers combine:

- **Initial scheme** if target cooperative;
- **Fallback squeeze-out** if scheme fails;
- **Pre-takeover** documentation contingency planning.

### Listed-company specifics

For LSE-listed Jersey TopCos:

- **UK Takeover Code** applies (UK Code applies to Jersey
  companies if listed in UK);
- **Takeover Panel** oversight;
- **Specific procedural** requirements;
- **Jersey company law** as underlying corporate law.

### Cross-border M&A

For cross-border Jersey-target M&A:

- **Foreign acquirer** considerations alongside Jersey
  procedures;
- **Acquirer-jurisdiction** law on consideration shares (if
  any);
- **Tax structuring** for both sides.

## What to do next

For an active M&A transaction:

1. **Choose** route based on factors above;
2. **Engage** Jersey counsel;
3. **Coordinate** with takeover-code / regulatory advisers;
4. **Plan** timeline carefully;
5. **Execute** with proper procedural compliance.

## Pitfalls

- **Wrong route**: squeeze-out attempted without 90%
  achievable;
- **Class-composition** litigation in scheme;
- **Procedural** failures (notice, deadlines);
- **Cross-border timing** mismatches.
