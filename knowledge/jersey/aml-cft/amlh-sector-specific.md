# AML/CFT Handbook — Sector-Specific Application

The Handbook applies across all
[Schedule 2 supervised businesses](./schedule-2-businesses.md)
but its application is **sector-specific** — the risks,
typologies, and operational arrangements differ enough that
the Handbook contains (or supplements with) sector-specific
guidance for each major regulated activity.

This file is the section index — one sub-section per sector.
Each sub-section file documents the Handbook's sector-specific
application alongside the relevant
[Code of Practice](../financial-regulation/codes-of-practice.md).

## Cross-sector themes

Across all sectors:

- The **risk-based approach** ([`./amlh-risk-assessment.md`](./amlh-risk-assessment.md))
  applies, but the **risk taxonomy varies** — a TCB sees
  different typology mix than an MSB.
- **PEP exposure** is more concentrated in TCB / IB / private
  banking; **cash typology** is more concentrated in MSB and
  some DNFBPs.
- **Beneficial-ownership** complexity is the central CDD
  challenge for TCB / FSB / company-services; the
  identification-and-verification challenge is more
  individual-focused for IB / banking.
- **Sanctions exposure** is broadest in banking and MSB
  (every payment is a sanctions touchpoint).
- **VASPs** are treated as inherently high-risk under FATF
  / JFSC framing — see the dedicated sub-section.

## Wealth-management cross-sector view

A Jersey wealth-management group typically holds **multiple
licences** — at minimum TCB and Investment Business, often FSB
as well. The Handbook's sector-specific application means the
group must operate **harmonised but sector-appropriate**
AML/CFT controls — the same BWRA framework, the same MLRO,
but sector-tailored customer-risk methodology and EDD trigger
sets.
