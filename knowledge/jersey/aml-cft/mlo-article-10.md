# MLO 2008 Article 10 — Ongoing Monitoring

Article 10 requires **ongoing monitoring** of the business
relationship.

## Components

Ongoing monitoring includes:

### Transaction monitoring

- **Compare** transactions against expected pattern
  (Article 9);
- **Flag** unusual / anomalous transactions;
- **Investigate** flagged transactions;
- **Escalate** to MLRO where unresolved;
- **System-supported** monitoring tools.

### Periodic CDD refresh

- **Identity information** current and accurate;
- **Beneficial ownership** still as recorded;
- **Sanctions / PEP** status current;
- **Source of funds / wealth** consistent with expected.

### Trigger-based review

- **Adverse media**;
- **Customer changes** (address, occupation, structure);
- **Transaction anomalies**;
- **Risk score** changes.

### System-supported monitoring

- **Transaction monitoring** tools applying rules / models;
- **Sanctions screening** systems;
- **Adverse media** monitoring;
- **Behavioural analytics** for higher-risk relationships.

## Calibration

Monitoring is **risk-based**:

- **High-risk** relationships: more frequent review, lower
  alert thresholds, more analyst attention;
- **Standard** relationships: automated monitoring with
  periodic review;
- **Low-risk** relationships: automated only, with periodic
  refresh.

## Documentation

- **System outputs** — flags, alerts, dispositions;
- **Investigator notes** on flagged matters;
- **Resolutions** — false positive, escalate, SAR;
- **MLRO oversight** of monitoring effectiveness.

## CDD refresh cycle

Typical refresh cycles:

- **High-risk** customers: **annually** or more frequently;
- **Medium-risk**: every **2-3 years**;
- **Low-risk**: every **3-5 years**;
- **Trigger events** override the schedule.

## JFSC expectations

The JFSC expects:

- **Documented** monitoring methodology;
- **Effective** transaction monitoring systems;
- **Proportionate** to size and risk;
- **Senior management** oversight of monitoring program;
- **Specific** recent themes (PEP monitoring, sanctions,
  high-risk jurisdictions).
