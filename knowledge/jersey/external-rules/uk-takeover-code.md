# UK Takeover Code — Jersey Interaction

**This is a bridge file.** It documents how the UK City Code on
Takeovers and Mergers ("**the Code**") applies to **Jersey-
incorporated companies** and where it interacts with Jersey
company law. **For the substance of any Code Rule, cite the
Takeover Panel's current published Code directly** — not this
file. The Code is updated periodically and only the Panel's
publication is authoritative.

## The Panel and the Code

The **Panel on Takeovers and Mergers** ("the Panel") is the UK
non-statutory body that issues and administers the **City Code on
Takeovers and Mergers** — the rule-book governing public-company
takeovers and certain other transactions involving companies
with shares traded on UK markets and on certain other markets.

The Code's stated objective is **fair and equal treatment of
shareholders** in the target of an offer; it is not concerned
with the commercial advisability of an offer. The Code operates
on a **principles-plus-rules** structure: six General Principles
and a body of detailed Rules. Code interpretations are issued by
the Panel as Panel Statements and Practice Statements.

## When the Code applies to a Jersey-incorporated company

This is the **load-bearing applicability test** for any
Jersey-incorporated bid target or bidder.

The Code applies to offers for a company that has its registered
office in the **United Kingdom, the Channel Islands or the Isle
of Man** **and** whose securities are admitted to trading on a
UK regulated market (the LSE Main Market), a UK multilateral
trading facility (AIM, AQSE Growth Market), or a stock exchange
in the Channel Islands or the Isle of Man (TISE).

For a **Jersey-incorporated** company, the practical position is:

- **Yes**, the Code applies, if the company is admitted to trading
  on the LSE Main Market, on AIM, on AQSE Growth Market, or on
  TISE. This is the typical position for a Jersey-incorporated
  plc using Jersey as a TopCo for a UK listing — see
  [`../use-cases/founder-entrepreneur/why-jersey-topco.md`](../use-cases/founder-entrepreneur/why-jersey-topco.md)
  and the worked example
  [`../use-cases/founder-entrepreneur/worked-example-jersey-topco-uk-listing.md`](../use-cases/founder-entrepreneur/worked-example-jersey-topco-uk-listing.md).
- **No**, the Code does not apply, if the company has Jersey
  registered office but **no relevant market admission** — a
  private Jersey company is not a Code target.

A **Jersey-incorporated bidder** is bound by the Code in the
same way as any other Code bidder if it is making an offer for
a Code target.

**Verify the applicability test against the Panel's current Code
Section 3 (Companies, transactions and persons subject to the
Code)** before stating the position authoritatively in any
specific transaction. Section 3 is the canonical source.

## The six General Principles

The Code's General Principles are stated at a high level and
apply to every transaction in scope. They cover:

1. **Equal treatment of shareholders** of the same class;
2. **Sufficient time and information** for shareholders to
   reach a properly informed decision;
3. **Board acting in the interests of the company as a whole**
   and not denying the holders of securities the opportunity to
   decide on the merits of the bid;
4. **No false markets** in securities of the offeree, the
   offeror or any other company concerned;
5. **Offeror's certainty of funds** before announcing an offer;
6. **No frustrating action** by the target without shareholder
   approval once a bona fide offer is communicated.

The detailed Rules operationalise the Principles. Where a Rule
gap exists, the Principle still binds.

## The load-bearing Rules for Jersey practitioners

Three Rules come up repeatedly in Jersey-touching transactions
and an agent should know which file to send the user to (the
Panel's published Rule, not this bridge file):

### Rule 9 — Mandatory offer threshold

When a person acquires, alone or in concert, **30%** or more of
the voting rights in a Code-subject company, or when a person
already holding between 30% and 50% acquires more shares
carrying voting rights, Rule 9 obliges the acquirer to make a
**mandatory cash offer** (or cash-equivalent) for all the
remaining shares of the same class at the highest price paid in
the preceding 12 months.

The 30% threshold and the mandatory-offer mechanism are the most
practically important Rule for Jersey-incorporated plcs with
concentrated shareholdings.

### Rule 21 — Frustrating action

Once a bona fide offer has been communicated to the target's
board, or the target's board has reason to believe a bona fide
offer is imminent, **the target's board may not take any action
that might frustrate the offer** — issuing shares, disposing of
material assets, entering into material contracts outside the
ordinary course — without **shareholder approval** in general
meeting.

Rule 21 sits alongside the Jersey directors' fiduciary duty in
[`../companies/directors-duties.md`](../companies/directors-duties.md)
under Article 74 CJL 1991. The Code's restriction is procedural
(requiring shareholder vote); the CJL 1991 duty is substantive
(act in good faith in the best interests of the company). Both
must be satisfied.

### Squeeze-out and sell-out — Code interface with CJL 1991

After a successful offer, the bidder may seek to compulsorily
acquire any minority shares not tendered. The Code addresses
this through Rule 36 and the surrounding squeeze-out procedure;
the substantive Jersey-law squeeze-out mechanism is in
**Articles 117–122 of the CJL 1991**, covered in
[`../companies/squeeze-out.md`](../companies/squeeze-out.md) and
[`../companies/sell-out.md`](../companies/sell-out.md). The two
work together: the Code governs the offer mechanics; CJL 1991
governs the statutory compulsory-acquisition outcome.

## Schemes of arrangement as an alternative deal structure

A bid for a Code-subject Jersey-incorporated company can be
structured as a contractual offer (the Code's default
contemplation) **or** as a Jersey-statutory **scheme of
arrangement** under Articles 125–127 of the CJL 1991 — see
[`../companies/scheme-of-arrangement.md`](../companies/scheme-of-arrangement.md).
The scheme route involves the Royal Court sanctioning a court-
approved arrangement; the Code applies to the scheme but with
modifications (Appendix 7 to the Code historically addressed the
scheme overlay; consult the current Appendix before relying on
specifics).

The choice between contractual offer and scheme has commercial
consequences (timing, voting thresholds, stamp duty, certainty)
that depend on the specific facts. The agent should surface
both routes when an applicable question lands.

## Irrevocables and letters of intent

Before announcing an offer, a bidder typically secures
**irrevocable undertakings** from large target shareholders to
accept the offer. The Code's Rule 2.7 announcement requirements
and Rule 19 information disclosure interact with the form of
irrevocables (hard, soft, hard-with-carve-outs) the bidder
takes. The Code requires disclosure of irrevocables and their
terms in the announcement and offer document.

This is process detail an M&A counsel will know. The bridge
file flags that the Code is what governs it; the substantive
Rules live in the Panel's publication.

## Interaction with the Jersey Takeovers and Mergers Panel

Jersey has its own **Companies (Takeovers and Mergers Panel)
(Jersey) Law 2009** establishing a Jersey-domestic Takeovers
Panel — see
[`../companies/takeovers-panel-2009-index.md`](../companies/takeovers-panel-2009-index.md).
The Jersey Panel's role in transactions to which the UK Code
applies is largely supportive: the Jersey Panel exists for
Jersey-internal takeover situations and supplements (it does not
replace) the UK Panel's role on transactions within UK Code
jurisdiction.

The two-panel structure occasionally surfaces choice-of-forum
and concurrent-jurisdiction issues that benefit from early
practitioner advice. The Jersey-domestic statute file (when its
section files are upgraded from stub) is the canonical place to
land for the Jersey side; this bridge file is the canonical
landing for the UK side.
