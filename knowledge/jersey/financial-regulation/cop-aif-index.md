# Code of Practice for Alternative Investment Funds — Index

Agent-facing entry point for the **JFSC Alternative Investment
Funds Code of Practice** (current version effective **16 April
2026**), the binding conduct standard for AIFs in Jersey under
the Jersey AIF regulatory framework, including the AIF (Jersey)
Regulations 2012 and the related JFSC supervisory regime.

The 16 April 2026 effective date is **aligned with AIFMD II's
EU application date** — see
[`../../frontier/aifmd-ii-april-2026.md`](../../frontier/aifmd-ii-april-2026.md).
The interaction between the AIFMD II framework, Jersey's NPPR
positioning, and this updated Code is itself an open practitioner
question worth tracking.
