# Code of Practice for General Insurance Mediation Business — Index

Agent-facing entry point for the **JFSC Code of Practice for
General Insurance Mediation Business** (current version
effective **1 January 2021**), the binding conduct standard for
Jersey businesses registered to carry on general insurance
mediation (broking, intermediation) under the
[Financial Services (Jersey) Law 1998](./financial-services-jersey-law-1998.md).

For the **GIMB regulated activity class** perimeter read
[`fsl-class-gimb.md`](./fsl-class-gimb.md). For the insurance
**underwriting** Code see
[`cop-insurance-index.md`](./cop-insurance-index.md) — a
separate Code applies to insurers themselves.
