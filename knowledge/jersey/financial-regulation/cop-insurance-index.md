# Code of Practice for Insurance Business — Index

Agent-facing entry point for the **JFSC Code of Practice for
Insurance Business** (current version effective **1 January
2021**), the binding conduct standard for Category A and
Category B permit-holders under the
[Insurance Business (Jersey) Law 1996](./insurance-business-law-1996.md).

For the underlying statutory framework — Category A
(non-captive / commercial) and Category B (captive) permits,
insurer authorisation conditions, and supervisory powers — read
[`insurance-business-law-1996.md`](./insurance-business-law-1996.md)
and the per-Article files in
[`../insurance/`](../insurance/index.md).
