# Code of Practice for Certified Funds — Index

Agent-facing entry point for the **JFSC Certified Funds Code of
Practice** (current version effective **1 January 2021**), the
binding conduct standard for certified funds in Jersey —
distinct from the Code that applies to fund **service providers**
(see [`cop-fsb-index.md`](./cop-fsb-index.md)).

This Code applies at the **fund level**: governance of the fund,
investor disclosure, the relationship with the fund's service
providers, and compliance obligations under the
[Collective Investment Funds (Jersey) Law 1988](./collective-investment-funds-law-1988.md)
and (where applicable) AIFMD-era regulation.

The Code carries **five schedules** (per the JFSC publication
page); these are recorded as separate stub files below.
