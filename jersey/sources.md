# Canonical Sources for Jersey

Primary-source URLs used throughout the Jersey corpus. Files cite these
inline; this page exists so a reviewer (or the agent itself) can validate
that we are pointing at the right roots.

When citing from a specific page below, prefer linking the concrete
sub-page in the file's frontmatter `sources` list. Reserve this list for
the section roots.

## Updating this list

root to this table if it isn't already here. When you discover that an
official URL has moved or been replaced, update both this file and the
sources block in every file that used the old URL.
