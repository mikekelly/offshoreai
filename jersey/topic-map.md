# Jersey Knowledge Base — Topic Map

## Purpose

This topic map provides **cross-section navigation** for the
Jersey corpus — showing how the various sections relate to
each other and which combinations of files address common
multi-disciplinary issues.

The corpus has ~25 directories and 1,000+ files. The
**section index files** (e.g., `trusts/index.md`) are the
**vertical** entry points; this map is the **horizontal**
navigation.

   ([tax](tax/index.md));
